HdBuy is a way to easily find and bookmark resale flats of your liking.

Report bugs to: hdbuy@gmail.com
GitHub page: <https://github.com/AY2021S2-CS2113-F10-1/tp>
User Guide: <https://github.com/AY2021S2-CS2113-F10-1/tp/blob/master/docs/UserGuide.md>

Available commands:
===============================================
    filter <attribute> <value>                           Add a filter condition. eg: filter location woodlands
                          list    Show all currently set filter condition to filter units matching preferences
                         clear                                      Remove all currently set filter conditions
                          find                             Search for units with the current filter conditions
              sort <direction>                 Sort search results in ascending(asc) or descending(desc) order
                  save <index>                             Add the unit at the inputted index to the shortlist
                remove <index>                        Remove the unit at the inputted index from the shortlist
                     shortlist                                                 Show all units in the shortlist
                          exit                                                            Exit the application
